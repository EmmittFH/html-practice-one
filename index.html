Lists

There are lists that can be created in HTML. There is an ordered list and an unordered list. An ordered list utilizes numbers. This can be used for steps in a sequential order. An unordered list utilizes bullet points. This can be used for a grocery list of items to buy.

This is an ordered list example:

  1. Wake up
  2. Brush teeth
  3. Shower
  4. Eat Breakfast
  5. Go to work
  6. Take nap after work
  7. Go for a walk
  8. Shower after walk
  9. Eat dinner
  10. Learn to code
  11. Get ready for bed and rest

This is an example of an unordered lists:

  • Get an oil change for the car
  • Go to Target
  • Drive to the movies and watch a movie
  • Buy a Braves baseball cap

Ordered lists and unordered lists can be used together:

  1. Elementary School
    • Grades:
      • K-5
  2. Middle School
    • Grades:
      • 6-8
  3. High School
    • Grades:
      • Freshmen
      • Sophomore
      • Junior
      • Senior
